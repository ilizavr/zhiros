> web socket - протокол созданный для создания двунаправленного асинхронного соединения между клиентом и сервером. находится на прикладном(4 уровне) стека TCP/IP и на прикладном(7 уровне) стека OSI
>  протокол web socket для создания соединения использует протокол tcp
>  если tcp - интерфейс с помощью которого протоколы прикладного уровня обращаются к транспорному, то web socket в свою очередь является протоколом который описывает правила взаимодействия на прикладном уровне между хостами
