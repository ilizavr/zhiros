# my susspose about gencc tool
i've thought new software after compiler: gencc. What is it? it's tool for fast writing proggrams in C. How it works? you writing .genc file simple syntax,invoke genc compiler and getting C code. example of genc proggram:

```python
import stdio.h

def RED "\e[31m"
def RES "\e[0m"

fun just(int s)
{
if s <= 5
{
stdio.printf(RED "s variable has critical damage."RES "Abort")
for i := 0;i<1000000;i++
{
__asm__("nop")
}
__asm__("dup2")
}
return
}
fun main(int argc,char** argv) (int)
{
// __ in this context: just current package. 
__.just(argv[argc-1])
return 0
} 
```

invoke gencc compiler:

```bash
gencc example.genc -s . -o example.c -w
```

see example.c file:

```c

```

or,you can see assembly form(but gencc use its own assembly):
```bash
gencc example.genc -s . -S example.s
```

and see example.s,that has gencc assembly(i'm to write only small part of just func):
