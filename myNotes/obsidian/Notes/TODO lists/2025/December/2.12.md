# API of my auth service
## on go
```go
Register(ctx context.Context, creds Credentials) error  

Authenticate(ctx context.Context, creds Credentials) (Session, error)

Authorize(ctx context.Context, sessionID string) (UserInfo, error)
```
## on proto

```protobuf
syntax = "proto3";

 option go_package = ".;auth";

package auth;

import "google/protobuf/empty.proto";

service AuthService{
rpc Register(Credentials) returns (google.protobuf.Empty);
rpc Authentificate(Credentials) returns (Session);
rpc Authorize(Session) returns (google.protobuf.Empty);
}

message Credentials{
string name = 1;
string password = 2;
}

message Session{
string  name = 1;
string  uuid = 2;
bool  is_admin = 3;
int64  timestap = 4;
}
```


# compiling proto code in go code
```shell
protoc \
  --go_out=. \
  --go_opt=paths=source_relative \
  --go-grpc_out=. \
  --go-grpc_opt=paths=source_relative \
  auth.proto
```
