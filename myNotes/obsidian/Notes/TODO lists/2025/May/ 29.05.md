
# Функции нужные для реализации ядра RPS

## функция GetUserChoice
функция GetUserChoice с сигнатурой:
```go
func  (r *RPS) GetUserChoice(reader io.Reader) (string,error){...}
```

задача функции GetUserChoice  заключается в том, чтобы прочитать данные из обьекта r и вернуть прочитанные данные. пример:

```go
func main(){

choice,err := r.GetUserChoice(os.Stdin)

if err != nil{
fmt.Println(err)
}

fmt.Println("выбор:"choice)
}
```

вывод:
```go
Добро пожаловать в игру камень, ножницы, бумага! Правила: камень побеждает ножницы, ножницы побеждают бумагу, бумага побеждает камень. Введите стоп чтобы остановить игру:
камень

выбор: камень
}
```

именно по такому принципу и должна работать функция GetUserChoice 


## функция GenerateComputerChoice
функция должна сгенерировать число находящийся в пределах длины слайса фигур. после генерации числа функция должна взять фигуру из массива с индексом числа который был сгенерирован, и вернуть фигуру. сигнатура:

```go
func (r *RPS) GenerateComputerChoice() FigureRPS {...}
```

пример использования:
```go
func main(){
fig := r.GenerateComputerChoice()
fmt.Println("фигура:",fig)
}
```

1 вывод:
```go
фигура: "камень"
```

2 вывод:
```go
фигура: "камень"
```
3 вывод:
```go
фигура: "бумага"
```

### как изменить код чтобы функция работала?
1. добавить тип ```FigureRPS``` в код. это просто мапа строки и слайса строк, тоесть тип будет выглядеть так:
   ```go
	  type FigureRPS map[string][]string
    ```
    это нужно для уменьшения дублирования кода, чтобы не писать каждый раз ```map[string][]string```. также это нужно для следующего действия 
2. изменить тип FiguresConfig чтобы  у него не было полей кроме слайса фигур. он должен выглядеть примерно так:
 ```go
 type FiguresConfig struct{
 Figures []FigureRPS `yaml:"figures"` // хранит  все фигуры которые есть в игре
}
```
это позволит легко итерироваться и генерировать рандомные фигуры.

3. сделать саму функцию GenerateComputerChoice. Следовать [[#функция GenerateComputerChoice]]

## функция  DetermineWinner
функция DetermineWinner занимается определением победителя. Вот сигнатура:
```go
func (r *RPS) DetermineWinner(usrChoice string,CompChoice FigureRPS) int
```

функция принимает 2 параметра: выбор игрока и выбор компьютера. компьюттер сравнивает слабости с помощью  слайса слабостей  и определят победителя. 
1 - победил пользователь(константа USERWIN)
2 - победил компьютер(константа COMPWIN)
-1 -  ничья(константа STANDOFF)
