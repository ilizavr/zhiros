**что узнал нового**
можно сделать визуальное отображение покрытия тестов, возможность эта появляется с помощью ```go tool cover``` 


ключ -coverprofile в go test  занимается записыванием метрик о покрытии теста в файл.  как аргумент функция принимает имя  файла куда можно будет записать нужные метрики

после генерации файла метрики теперь можно визуализировать покрытие теста

для этого в go tool cover нужно указать ключ html  который генерирует html представление текстового файла метрик, в общем случае алгоритм выглядит вот так:
```bash
go test -coverprofile=cover.out # генерируем файл метрик покрытия кода на основе компиляции тестов 

go tool cover -html=cover.out #генерируем html документ созданный на основе файла метрик покрытия кода. ВАЖНО! браузер откроется автоматически а значит комнда разовая, чтобы 2 раз посмотреть визуальное покрытие кода нужно заново генерировать html представление файла метрик
```
