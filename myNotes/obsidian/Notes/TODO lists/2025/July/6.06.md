# а что мокать то?

мы мокаем  функцию для вытягивания user_id из jwt токена.  мок должен быть через  нтерфейс:

```go
type JWTPuller interface{
Pull(token string) (string,error)
}
```

должна быть структура:

```go
type JWTUserIdPuller struct{
user_id string
}

func (j *JWTUserIdPuller) Pull(token string) (string,error){
//..  какая то реальная логика вытягивания
}
```


функцию refreshTokens мы оставим как есть, но мы должны изменить функцию  CreateNewSignedJwtTokens(userId string) (string, string, error). ее нужно превратить в метод интерфейса. вот конкретно:

```go
type TokenGenerator interface{
GenerateTokens(userId string) (string,string,error)
}
```
