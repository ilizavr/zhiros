**что узнал за время чтения документции**
- есть удобный пакет  [telegoutil](https://pkg.go.dev/github.com/mymmrac/telego@v1.1.1/telegoutil) который позволяет меьшне писать что удобно
- у [telegoutil](https://pkg.go.dev/github.com/mymmrac/telego@v1.1.1/telegoutil) есть функция Message для подготовки сообщения  к отправлению
- у [telegoutil](https://pkg.go.dev/github.com/mymmrac/telego@v1.1.1/telegoutil) есть функция InlineKeyboard которая создает слайс inlie клавиатур 
- у [telegoutil](https://pkg.go.dev/github.com/mymmrac/telego@v1.1.1/telegoutil) есть функция [WithReplyMarkup]() которая позволяет  добавить к Message дополнительную разметку(например inline клавиатуру)
- пакет [telegoutil](https://pkg.go.dev/github.com/mymmrac/telego@v1.1.1/telegoutil) имеет  функцию [InlineKeyboardButton]()  для создания отдельной кнопки клавиатуры, данная функция возращает  inline кнопку и у данной кнопки можно вызвать метод [WithCallbackData]() для создания имени callback запроса
