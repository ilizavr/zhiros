# конспект по mermaid
в основном используй graph TD чтобы создать граф

- для создания узла и  описание для него нужно указать его имя,  например A и в квадратных скобках описание. пример: A[ты тупой]
- для создания синопсиса между узлами, используй -->. пример A[ту тупой] --> B[да]
- для создания описания синопсиса используй синтаксис -- **описание**  перед созданием синопсиса. пример: B -- да --> D[иди нафиг]
- для создания ромбовидного узла перед описанием используй не квадратные скобки а фигурные. Пример: B{foo!}
