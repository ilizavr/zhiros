# что значит производный класс
допустим есть интерфейс(или класс в java) computer:
```go
type Computer interface{
Computing(command mWord) Result
}
```

который обозначает любое вычислительное устройство. а есть класс(или структура в go) phone:

```go
type Phone struct{
CPU string
GPU string
}
func (p *Phone) Computing(command mWord) Result{
...
}

```

класс Phone является производным от класса Computer, ведь Phone имеет методы базового класса. проще говоря, Phone это производный от клсса Computer класс, тоесть это значит что Phone унаследовал базовые методы от Computer
