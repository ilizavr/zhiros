# dup2 function

**Purpose:**

- `dup2()` duplicates an existing file descriptor (`oldfd`) to an specified file descriptor (`newfd`).
- If `newfd` is already open, it is **closed** before being reused.
- After calling `dup2()`, `newfd` becomes an **alias** for `oldfd`.

**Prototype:**

```c
int dup2(int oldfd, int newfd);
```

**Return Value:**

- On success: returns `newfd`.
- On error: returns `-1` and sets `errno` with that error,executed in dup2 exec.
