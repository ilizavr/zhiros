# structure of build byte of 0x1F6 port
### 8 bits of 0x1F6:

text

Bit 7 | Bit 6 | Bit 5 | Bit 4 | Bit 3 | Bit 2 | Bit 1 | Bit 0
------|-------|-------|-------|-------|-------|-------|-------
  1   |  LBA  |   1   | DRV   |      LBA bits 24-27
      |  mode |       | sel   |      (or CHS head bits)

### Bit by bit:

**Bit 7 (0x80)** — Always 1

- Historical, must be set
    

**Bit 6 (0x40)** — LBA Mode

- `1` = Use LBA addressing (what you want)
    
- `0` = Use CHS addressing (old crap)
    

**Bit 5 (0x20)** — Always 1

- Another historical bit, just set it
    

**Bit 4 (0x10)** — Drive Select

- `0` = Master device
    
- `1` = Slave device
    

**Bits 3-0 (0x0F)** — LBA bits 24-27 (in LBA mode)

- These hold the highest 4 bits of your 28-bit LBA address
    
- In CHS mode, these bits hold the head number
