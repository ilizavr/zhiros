# example of divide 14:7 in x86_64
```asm
mov rax, 14   ; делимое
xor rdx, rdx  ; обнулить старшую часть
mov rbx, 7    ; делитель
div rbx       ; RDX:RAX / RBX → частное в RAX, остаток в RDX
; Теперь RAX = 2, RDX = 0
```
