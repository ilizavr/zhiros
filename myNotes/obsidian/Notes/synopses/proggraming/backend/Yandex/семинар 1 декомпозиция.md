# Debezium - определение

- Debezium - прогграма с open source, которая читает из бд, и переносит данные куда то в другое место, например в кэш бд или любую другую no sql

- connector -  единица в debezium, которая читает логи транзакции из СУБД, и отправляет их в другое место, например в брокер. работает на сокетах(мое предположение которое скорее всего верно)

prometeus - база данных для метрик
graphana - интерфейс для prometeus чтобы смотреть метрику в  удобноваримом виде


в семинаре будет практика с prometeus
