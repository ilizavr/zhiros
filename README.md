## 🚀 Как запустить

Для успешного запуска жирОС выполните следующие шаги в терминале:

### 1. Подготовка окружения
Перед первым запуском необходимо создать рамдиск (Ramdisk):
`make create_rd`

### 2. Компиляция и запуск (Основной способ)
Сборка проекта и запуск в эмуляторе QEMU с поддержкой GRUB и графического фреймбуфера:
`make clean build build_grub run_grub`

---

## ⚠️ Возможные проблемы и их решение

Если у вас возникли проблемы с основным способом запуска, можно использовать альтернативную команду:

`make clean build run_qemu`

> **Важно:** При использовании этого метода **не будет работать фреймбуфер (framebuffer)**. Настоятельно рекомендуется разобраться с поддержкой GRUB для полноценного графического режима.
